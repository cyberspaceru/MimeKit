InternetAddress Class

An internet address, as specified by rfc0822.

Syntax

public abstract class InternetAddress : IComparable<InternetAddress>, IEquatable<InternetAddress>

Remarks

A MimeKit.InternetAddress can be any type of address defined by the original Internet Message specification.

There are effectively two (2) types of addresses: mailboxes and groups.

Mailbox addresses are what are most commonly known as email addresses and are represented by the MimeKit.MailboxAddress class.

Group addresses are themselves lists of addresses and are represented by the MimeKit.GroupAddress class. While rare, it is still important to handle these types of addresses. They typically only contain mailbox addresses, but may also contain other group addresses.

Returns

Remarks

If the encode parameter is true, then this method will return an encoded version of the internet address according to the rules described in rfc2047.

However, if the encode parameter is false, then this method will return a string suitable only for display purposes.
