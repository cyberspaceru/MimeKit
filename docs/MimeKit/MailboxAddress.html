MailboxAddress Class

A mailbox address, as specified by rfc822.

Syntax

public class MailboxAddress : InternetAddress

Remarks

Represents a mailbox address (commonly referred to as an email address) for a single recipient.

IsInternational Property

Gets whether or not the address is an international address.

Syntax

public bool IsInternational { get; }

Value

true if th address is an international address; otherwise, false.

Remarks

International addresses are addresses that contain international characters in either their local-parts or their domains.

For more information, see http://tools.ietf.org/html/rfc6532#section-3.2

ToString Method

Syntax

public override string ToString (bool encode)

Parameters

encode
If set to true, the MimeKit.MailboxAddress will be encoded.

Returns

A string representing the MimeKit.MailboxAddress.

Remarks

Returns a string containing the formatted mailbox address. If the encode parameter is true, then the mailbox name will be encoded according to the rules defined in rfc2047, otherwise the name will not be encoded at all and will therefor only be suitable for display purposes.
