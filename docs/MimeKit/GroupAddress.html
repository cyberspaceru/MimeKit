GroupAddress Class

An address group, as specified by rfc0822.

Syntax

public class GroupAddress : InternetAddress

Remarks

Group addresses are rarely used anymore. Typically, if you see a group address, it will be of the form: "undisclosed-recipients: ;".

Members Property

Gets the members of the group.

Syntax

public InternetAddressList Members { get; }

Value

The list of members.

Remarks

Represents the member addresses of the group. Typically the member addresses will be of the MimeKit.MailboxAddress variety, but it is possible for groups to contain other groups.

ToString Method

Syntax

public override string ToString (bool encode)

Parameters

encode
If set to true, the MimeKit.GroupAddress will be encoded.

Returns

A string representing the MimeKit.GroupAddress.

Remarks

Returns a string containing the formatted group of addresses. If the encode parameter is true, then the name of the group and all member addresses will be encoded according to the rules defined in rfc2047, otherwise the names will not be encoded at all and will therefor only be suitable for display purposes.
